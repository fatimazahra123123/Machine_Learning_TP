 Machine_Learning_TP
— Régularisation & Overfitting en Machine Learning

Ce notebook Jupyter explore les concepts fondamentaux de **régularisation**, 
d'**overfitting** et d'**underfitting** à travers des expériences visuelles 
avec `scikit-learn`.

 Objectifs

- Comprendre la différence entre underfitting, bon modèle et overfitting
- Appliquer les techniques de régularisation **Lasso (L1)** et **Ridge (L2)**
- Visualiser les frontières de décision selon la complexité du modèle
- Utiliser les pipelines scikit-learn pour structurer les expériences
